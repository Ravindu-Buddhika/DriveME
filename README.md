# 🚗 DriveME - Full Stack Car Rental System

DriveME is a comprehensive web application for managing car rentals, built using modern technologies. It includes a user-friendly frontend for customers and a powerful backend for administrative controls.

---

## 🏗️ Project Architecture
The project is split into two specialized repositories to maintain a clean **Decoupled Architecture**:

### 🔹 [Frontend Repository](https://github.com/Ravindu-Buddhika/car-rental-system-frontend)
* **Tech Stack:** React.js, Bootstrap, Axios.
* **Key Features:** Booking system, Vehicle browsing, Responsive UI.

### 🔹 [Backend Repository](https://github.com/Ravindu-Buddhika/car-rental-system-backend)
* **Tech Stack:** Java, Spring Boot, Spring Security (JWT), MySQL.
* **Key Features:** REST API development, Authentication, Database Management.

---

## 🛠️ How to Run Locally

### 1. Backend Setup
1. Clone the backend repository.
2. Create a MySQL database named `carmanagement`.
3. Update `application.yml` with your MySQL username and password.
4. Run the application via your IDE or Maven.

### 2. Frontend Setup
1. Clone the frontend repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to launch.
